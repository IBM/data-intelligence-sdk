.. _overview:

Overview
========

The ``IBM watsonx.data intelligence SDK`` is a comprehensive Python toolkit designed to support data intelligence operations across multiple domains. Built with a modular architecture, the SDK enables different teams to contribute specialized functionality while sharing common components.

This section provides an overview of the SDK's capabilities, recent updates, and important information for users.
